About

This app is an angular/firebase app that implements an easy way to track bugs across a company. It also integrates several new and updated technologies

  • angularJS: is what HTML would have been, had it been designed for building web-apps.
  • Firebase: Google's mobile platform that helps you quickly develop high-quality apps and grow your business.
  • Bootstrap 4: a sleek, intuitive, and powerful mobile first front-end framework for faster and easier web development.
  • ReactiveXa combination of the best ideas from the Observer pattern, the Iterator pattern, and functional programming
